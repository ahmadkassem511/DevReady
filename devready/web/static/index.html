Free AI parser (OpenRouter)

How to get a free key — takes about a minute, no credit card:

  1. Open openrouter.ai/keys and sign in.
  2. Click Create Key and copy it (it starts with sk-or-).
  3. Paste it below and click Save. It's stored only on this computer.
🔒 Your key never leaves your machine except to talk to OpenRouter directly. DevReady has no server and never sees it.

GitHub token (optional) — smoother Discover browsing

Discover loads popular projects from GitHub, which limits anonymous searches to about 10 per minute. A free token raises that a lot.

  1. Open github.com/settings/tokens → Generate new token (classic). No scopes needed — read-only public access is enough.
  2. Copy it and paste it below. It's stored only on this computer.
